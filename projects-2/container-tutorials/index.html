The "Container Tutorials" Project</h2>
 

In April 2007, when the use of IoC containers on the .Net Framework was in it's infancy I published a series of tutorials (14 in all) to guide developers through the basic concepts of using the Castle Windsor</a> to wire up services and implementations within your application.

The parts of the series can be found here:

  • Part 1 - Simple Configuration</a></li>
  • Part 2 - More Configuration (Arrays)</a></li>
  • Part 3 - Still More Configuration (Dictionaries)</a></li>
  • Part 4 - Switching Configurations</a></li>
  • Part 5 - Configuration Parameters</a></li>
  • Part 6 - Switching Between Lifestyles</a></li>
  • Part 7 - Switching Between Implementations</a></li>
  • Part 8 - Referencing implementations by key</a></li>
  • Part 9 - Constructor Injection</a></li>
  • Part 10 - Setter Injection</a></li>
  • Part 11 - Factories</a></li>
  • Part 12 - Decorators</a></li>
  • Part 13 - Injecting Service Arrays</a></li>
  • Part 14 - Startable Facility</a></li>
    </ul>

    These tutorials demonstrate configuration of windsor through an external XML configuration file - since the publish of these tutorials, XML based configuration has fallen largely out of favour, with most people opting to use programmatic configuration of their container, and convention over configuration - but the concepts outlined in this series remain largely unchanged.</div>

    As well as the original series, a number of other Authors ported the tutorials across to other IoC containers or configuration mechanisms:</div>

    • Ruprict's "Binsor Style" tutorials</a> (using the Binsor domain specific language for Windsor configuration)</li>
    • Michael McGuire's Unity Container Tutorials</a></li>
      </ul>
      </div>
