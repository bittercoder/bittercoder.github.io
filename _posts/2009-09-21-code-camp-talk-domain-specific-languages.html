The DSL Book</h2>
If you found this topic interesting and want to learn more about writing Domain Specific Languages with Boo, I strongly encourage you to buy Ayende's book</a> "Writing domain specific languages in boo".

</a> It's very thorough (350 odd pages) - and includes a lot of guidance around not only writing a domain specific language with Boo, but the far larger topic of incorporating a business DSL into real world applications, versioning concerns, batch compilation etc. that must be considered when ever you write a domain specific language to be used in a production system.

Do yourself a favour and buy it today</a> - it's currently in Mannings EAP (early access program) - but is complete and just undergoing the final editing stages before going to print.

              Syntax highlighting Editors</h2>
              I had a few questions via email / skype about the WPF syntax highlighting component I used in my demos - it was AvalonEdit, part of the SharpDevelop 4.0 project - you can grab the code via subversion from here:

              svn://svnmirror.sharpdevelop.net/sharpdevelop/trunk/SharpDevelop/src/Libraries/AvalonEdit</a>

              Ayende's book also covers building a simple script editor using the SharpDevelop windows forms editor control (know as ICSharpCode.TextEditor)... there is an article on codeplex here</a> which describes how to use that library.
